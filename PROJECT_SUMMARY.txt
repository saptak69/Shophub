MANGROVE: Platform Overview & Architecture
Mangrove is a high-contrast, brutalist e-commerce platform built specifically for underground streetwear, graphic apparel, and subverted fashion. Moving away from traditional, soft corporate layouts, the platform embraces a raw, terminal-inspired aesthetic utilizing massive typography and toxic neon accents.

Core Architecture & Tech Stack
The application runs on a full-stack JavaScript architecture designed for speed, security, and raw visual impact.

Frontend: Built with pure HTML5, modern CSS3 (utilizing CSS grid, flexbox, and backdrop filters), and vanilla ES6 JavaScript, intentionally avoiding heavy frameworks to ensure snappy transitions.

Backend: Powered by Node.js and Express.js, organized into a modular RESTful API architecture with distinct routes, database schemas, and security layers.

Database: Managed through MongoDB Atlas via Mongoose ODM, utilizing hardcoded, cached seeding logic to instantly deploy fresh, limited-run inventories.

Security & Auth: Features password hashing via bcryptjs and stateless session persistence using JSON Web Tokens (JWT) with restricted, admin-only routes.

Catalog & Streetwear Inventory
The storefront completely rejects generic, fast-fashion inventory. It is strictly curated for underground aesthetics, featuring premium heavyweight cotton drops and subverted classic pieces.

Inventory is categorized into three brutalist-aligned drops:

Anime Core: Heavyweight 240 GSM drop-shoulder tees and acid-washed fleece featuring dystopian, neo-metropolis art.

Y2K Graphic: Distressed knits, tactical tech-wear compression layers, and static-noise zip-up hoodies.

Limited Tactical: Parachute nylon cargo pants, aggressive titanium mesh runners, and destroyed old-money polos.

All product imagery is sourced from high-resolution content networks, styled to pop against absolute black and dark grey surfaces. Pricing is localized and scaled appropriately for the Indian market.

User Journey & Checkout Flow
The entire customer path has been optimized for high contrast and brutalist efficiency.

Discovery: Users are greeted by a high-impact hero section with a running marquee manifesto, allowing them to explore categories via a distinct bento-box masonry grid.

Catalog: The shop interface features sharp-edged product cards and stark filtering dropdowns, allowing users to effortlessly sort through drops.

Cart: A raw, two-column bag interface cleanly calculates subtotals, applies free shipping logic, and manages inventory quantities locally before conversion.

Terminal Checkout: The final step utilizes a terminal-style input form with harsh borders and neon accents, capturing shipping addresses and processing payment methods while updating inventory stock instantly in the database.

Operations & Account Management
Terminal Profiles: The account view serves as a secure operative terminal, cleanly rendering user details and transaction extraction histories in an uncompromising dark layout.

Admin Control Center: Administrators have access to an elevated operations surface where they can publish new apparel drops and track or update active transaction statuses without friction.

Mangrove stands as a fully weaponized, production-ready e-commerce experience, perfectly blending underground brutalist design with robust backend operations.
